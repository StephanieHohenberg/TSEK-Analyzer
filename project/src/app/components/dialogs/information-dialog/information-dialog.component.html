Über das Projekt

Der Prototyp TSEK-Analyzer wurde von Stephanie Hohenberg im Rahmen ihrer Masterthesis im Wintersemester 20/21 an der Freien Universität Berlin umgesetzt. Der Titel der Thesis lautet:

"Erste Schritte zu einer Taxonomie von Software-Engineering-Kontexten"

und beinhaltet das Akrononym, wofür TSEK steht.

Innerhalb ihrer Thesis hat Stephanie 129 technische Paper, welche auf der International Conference of Software Engineering (ICSE) 2020 veröffentlicht wurden, gelesen und analysiert. Um die Daten geeignet zu analysieren und um dadurch die aufgestellten Recherchefragen der Thesis beantworten zu können, wurde TSEK-Analyzer entwickelt. Dabei wurde ein User-Centered-Design Ansatz verfolgt.

 

 

 

Mehr Informationen können auf der Thesis-Seite gefunden werden

Das Projekt wurde mit dem Framework Angular umgesetzt und der Sourcecode, sowie weitere Artefakte, wie das Dokument zur Masterthesis selbst, ist auf Github öffentlich zugänglich.
